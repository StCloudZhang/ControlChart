What are Control Charts? Shewhart, 1924

Control charts separate common-cause variation (inherent to the process) from special-cause variation (assignable signals). Acting on common cause as if it were special cause makes processes worse — this is called tampering.

Control Limit Logic
General Form
Center Line ± 3σ̂

Limits are set at ±3 standard deviations of the plotted statistic — not the raw data. This gives approximately 99.73% coverage under normality, yielding a very low false-alarm rate.

X̄ Chart Subgroup Means

Each point is the mean of a subgroup (typically n = 4–5). Detects shifts in process average. Must always be paired with an R or S chart — the X̄ chart limits are derived from within-subgroup spread.

Reading the Chart

R Chart Subgroup Range

Monitors within-subgroup variability. The range (max − min) is simple to compute. For larger subgroups (n > 10) an S chart (standard deviation) is preferred. Always check the R chart before interpreting the X̄ chart — unstable spread invalidates the mean chart limits.

R Chart — Range of Subgroups
Parameters
Control Limits (n=5)
UCL = D₄ · R̄ (D₄ = 2.114)
LCL = D₃ · R̄ (D₃ = 0)

Individuals & Moving Range Chart n = 1

Used when only one observation is available per time period — common in slow processes (batch chemistry, monthly financials). The moving range between consecutive observations estimates short-term variability. Assumes approximate normality; can be sensitive to non-normal data.

p Chart Proportion Nonconforming

For attribute data: each item is either conforming or nonconforming. The p chart tracks the fraction defective per subgroup. Control limits are derived from the binomial distribution. Variable subgroup sizes produce variable-width limits — wider limits for smaller samples.

p Chart — Fraction Nonconforming
Parameters
Control Limits
UCL = p̄ + 3√(p̄(1−p̄)/nᵢ)
LCL = max(0, p̄ − 3√(p̄(1−p̄)/nᵢ))
Interpretation

c Chart Count of Defects · Fixed Area

Counts the number of nonconformities (defects) per inspection unit when the area of opportunity is constant — e.g., defects per printed circuit board, blemishes per car door panel. Based on the Poisson distribution.

c Chart — Defect Count per Unit
Parameters
Control Limits (Poisson)
UCL = c̄ + 3√c̄
LCL = max(0, c̄ − 3√c̄)
Interpretation

u Chart Defects per Unit · Variable Area

Like the c chart but for variable inspection areas — e.g., defects per 100 sq ft of fabric when roll lengths differ. The u statistic is defects ÷ inspection units. Limits vary by sample size, producing a ragged limit pattern.

u Chart — Defects per Inspection Unit
Parameters
Control Limits
u = c / n

LCL = max(0, ū − 3√(ū/nᵢ))
